## üéØ Quick Start Guide

### First Time Users

1. **Visit Landing Page**
   - Learn about MedAssist AI features
   - Click "Get Started" to create an account

2. **Complete Onboarding**
   - Watch the 3-slide tutorial
   - Learn how to upload reports and use AI features
   - Can skip if you prefer

3. **Accept Privacy Terms**
   - Review privacy notice before uploading
   - Understand data storage and deletion rights
   - Accept terms to proceed

### Uploading Your First Report

1. **Navigate to Dashboard**
   - Click "Upload New Report" button

2. **Review Privacy Consent** (First time only)
   - Read privacy policies
   - Check agreement boxes
   - Click "Accept & Continue"

3. **Upload File**
   - Drag and drop PDF or image
   - Or click to browse files
   - Supported formats: PDF, JPG, PNG
   - Max file size: 10MB

4. **Processing**
   - Watch AI extract data (OCR)
   - See real-time parsing progress
   - Takes 20-30 seconds

5. **View Results**
   - Automatically redirected to Analysis page
   - See all test results with explanations

## üÜò Troubleshooting

### Upload Issues
- **File too large**: Compress to under 10MB
- **Format not supported**: Convert to PDF/JPG/PNG
- **Privacy consent**: Must accept before first upload

### Processing Stuck
- Wait 30-60 seconds
- Refresh page if needed
- Re-upload if fails

### Can't Find Report
- Use search in History page
- Check filter settings
- Verify upload was successful

### Features Not Working
- Enable JavaScript in browser
- Clear browser cache
- Try different browser
- Check internet connection

### Common Questions

**Q: Is my data safe?**
A: Yes, all reports are encrypted and never shared. You can delete anytime.

**Q: Can I use this instead of seeing a doctor?**
A: No, always consult qualified healthcare professionals for medical decisions.

**Q: How often should I upload reports?**
A: Upload whenever you get new lab results to track trends.

**Q: What if I have critical results?**
A: The app will show a red alert banner. Contact your doctor immediately.

**Q: Can I share reports with my doctor?**
A: Yes, download the doctor-friendly PDF and bring to appointments.
